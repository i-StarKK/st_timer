# st_timer

**st_timer** is a lightweight timer system for RedM servers.
It allows developers to easily create and manage timers for gameplay systems, events, or scripts.

---

## Features

* Simple and lightweight
* Easy to integrate with other resources
* Optimized for performance
* Developer friendly

---

## Installation

1. Download or clone the repository.
2. Place the folder inside your server `resources` directory.
3. Add the resource to your server configuration:

```
ensure st_timer
```

4. Restart your server.

---

## Usage

This resource provides a simple timer system that can be used by other scripts.

You can modify or extend the system to fit your server's needs.

---

## Requirements

* Latest FXServer build
